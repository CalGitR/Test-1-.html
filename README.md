# Bill Split Calculator

A comprehensive web application for splitting bills among friends and roommates with ease.

**Team:** Raj, Prahlad, Saarthak

## Features

- ✅ Add multiple people to split with
- ✅ Add items/expenses with costs
- ✅ Assign items to specific people
- ✅ Automatic calculation of who owes whom
- ✅ Support for tax and tip calculations
- ✅ Transaction history
- ✅ Export results
- ✅ Clean, intuitive UI

## How to Use

1. Enter the names of people splitting the bill
2. Add expenses with descriptions and amounts
3. Assign each expense to one or more people
4. Add tax and tip percentages (optional)
5. View the settlement - see exactly who owes whom
6. Export or share the results

## Files

- `index.html` - Main application
- `styles.css` - Styling
- `script.js` - Calculator logic

## Getting Started

Simply open `index.html` in your web browser to start using the bill split calculator.
